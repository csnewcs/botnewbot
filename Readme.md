# BotNewBot
<img src="./temp%20logo.png" width="20%">

이 봇은 csnewcs가 만드는 디스코드 봇입니다.

## 기능
### 일반 사용자용
- 명령어
- 은행
- 도박
- 노래방

### 서버 관리자용
- 역할
- 처벌
- 처벌해제

## 현재 개발 중, 개발 예정
-  도박 고스톱(개발 중)
-  웹페이지(기발 중)
   -  서버 관리(관리자용)
   -  사용자들의 돈 관리(관리자용)
   -  여러 알림 채널 설정(관리자용)
   -  봇의 기능 활성화/비활성화(관리자용)
   -  서버 전체의 순위 보기
   -  모든 서버 중 내 서버의 순위 보기
- 메세지 관리
  - 욕 필터링
  -  한 번에 여러 개 메세지 삭제
  -  특정한 사람이 보낸 메세지 삭제
